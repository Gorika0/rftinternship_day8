# rftinternship_day8
# Employee Salary Insights using Pandas

## Project Overview
This project analyzes employee salary data using Python Pandas.  
It demonstrates important data analysis concepts such as:
- Grouping
- Aggregation using groupby()
- Average salary calculation
- Finding highest paid employees
- Employee counting
- Sorting data

This project is part of the Python Internship Day 8 Task.


# Concepts Used

- Pandas DataFrame=used to store and manage data in table format.
- groupby()=used to group similar data together department wise.
- mean()=used to calculate the average value.
- count()=used to count total records or employees.
- idxmax()=used to find the index of maximum value.
- sort_values()=used to sort the data in ascending or descending order.
